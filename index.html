Sprouting Spirit

UI/UX Dev & Level Designer

Sprouting Spirirt: Dream Duo is 3D puzzle platformer where you play as the main character, Maple, traversing her imaginary world with her imaginary friend Cornelius. Developed in the Unity game engine, we prided ourselves on creating our own tools to assist in the development process.

My capstone project, I worked as UI Developer / Level Designer and collaborated on a team of 16 other iterdisciplinary students. I developed the in-game UI Popup System. It main features include, controll over font size/slying, text formatting, appearance customizability such as color or use of custom images. Additionally flexibility of the popup’s fade in-out animation and speed

The total game experience consisted of 4 game levels, myself being responsible for overseeing the development of the 3rd level. This level being the point in which the puzzle difficulty beings to ramp up and serves to build up to the games final level.

Conway's Game of Life

Software Developer Backend

This web-based version of Conway’s Game of Life features a 25x25 tile board, controls to run a simulation, take the next step, pause, reset and randomize the simulation.

Starting out intially as a school project teach how to work with DOM elements, I want to expand it functionality to implement interactivity for each cell. In addition, to being able to chose an empty board to start with, users can now create their own patterns.
